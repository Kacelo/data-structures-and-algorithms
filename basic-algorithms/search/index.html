Linear Search

A linear search or otherwise known as the sequential search, is a method for finding an element within a list. This type of searching algorithms sequentially checks each element of the list until a match is found or until the whole list has been searched.

The search algorithm looks for the element that meet a certain condition. The algorithm will then traverse throughout the entire collection of items. If the desired element appears in the collection it will then a positive or true of the return type is a boolean, if it is not part of the collection it will display negative or false if the return type is a boolean.
