The minimum algorithm returns the element that is smallest in the collection. Firstly, the algorithm will then traverse throughout the entire collection of items. Then it will compare the first element with the next element in the collection. If the first element is smaller it will be considered to be the minimum until another element that is smaller appears in the collection. After all this the minimum element will be displayed.
